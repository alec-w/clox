# clox
Building an interpreter for a toy programming language

Follwong the free online book Crafting Interpreters at http://craftinginterpreters.com/ - well worth checking out if you're interested. in looking at how internals of a language are built.

## Requirements
1. gcc (tested on version 7.3.0)
2. gnu make (tested on version 4.1)
3. tested on Ubuntu 18.04

## Build and run
Makefile is in src directory.
```
cd src
make clean
make clox
../bin/clox
```
